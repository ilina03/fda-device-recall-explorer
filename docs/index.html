FDA Classification

Understanding the risk tiers

Per 21 CFR 513 — the foundation of FDA's device recall system.

Class I

Most Serious

Reasonable probability of causing serious adverse health consequences or death. Triggers urgent field actions and public notification.

Class II

Moderate Risk

May cause temporary or medically reversible adverse health consequences. Unlikely to cause permanent injury.

Class III

Low Risk

Unlikely to cause adverse health consequences, but still in violation of FDA-administered laws.

Quickstart

Run it locally in 4 steps

Python 3.10+ required. No API key or environment variables needed.

bash 
# Clone and enter the repo
$ git clone https://github.com/your-username/fda-device-recall-explorer.git
$ cd fda-device-recall-explorer

# Create and activate a virtual environment
$ python -m venv .venv && source .venv/bin/activate
$ pip install -r requirements.txt

# Launch
$ streamlit run app.py
# Opens at http://localhost:8501
